Class:            Cluster

Environment:      Siren
 Superclass:         Siren.EventGenerator
 Category:         Music-EventGenerators

Cluster is an abstraction of a set of simultaneous (or same-pitch) events.
A cluster need only have a set of pitches, or a rhythm.
